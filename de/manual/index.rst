Benutzerhandbuch
================

Leaflet ist eine freie JavaScript-Bibliothek, mit der WebGIS-Anwendungen erstellt werden können. Die Bibliothek verwendet HTML5, CSS3 und unterstützt somit die meisten Desktop- und Mobil-Browser.

Mit Contao-Leaflet können einfach Web Map Tile Services zusammen mit eigenen Geodaten mit Contao präsentiert werden. Die Geodaten können beispielsweise aus GeoJSON-Dateien geladen werden und mit interaktiven Funktionen wie Pop-ups versehen werden.
